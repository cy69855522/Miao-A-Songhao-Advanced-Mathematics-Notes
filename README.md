## P39 不定积分的性质
- d代表无穷小的△（增量）：dx = x1 - x2， d → 0
- `∫[f(x) + g(x)]dx = ∫f(x)dx + ∫g(x)dx`
- `∫kf(x)dx = k∫f(x)dx`
## P40 第一类换元积分法
- `∫f(φ(x))φ'(x)dx = ∫f(φ(x))dφ(x) = F(φ(x)) + C`
- 把d前面的某一部分（因子），求原函数，拿到d的里面去
- d的里面可以任意加减常数
## P41 第二类换元积分法
